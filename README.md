Профессиональный тюнер для гитары с автоматическим определением нот через микрофон.
Поддерживает стандартный строй, альтернативные строи, визуальную индикацию отклонения и выбор типа инструмента.

🧠 Концепция
GuitarTuner Pro — это интерактивный тюнер для настройки гитары, использующий микрофон для определения высоты звука. Он позволяет:

✅ Захватывать звук с микрофона в реальном времени.

✅ Анализировать частоту с помощью алгоритмов цифровой обработки сигналов.

✅ Определять ноту и отображать её название (E, A, D, G, B, E).

✅ Показывать отклонение от эталонной частоты в центах (графическая стрелка или индикатор).

✅ Поддерживать стандартный строй (EADGBE) и альтернативные строи (Drop D, Open G, DADGAD).

✅ Выбирать тип инструмента (гитара 6-струнная, бас-гитара, укулеле, скрипка).

✅ Визуализировать результат — цветной индикатор зелёный (в строю), жёлтый (близко), красный (далеко).

✅ Работать как в GUI-режиме (с окном), так и в консольном (с текстовым выводом).

🚀 Как запустить
Для работы требуется микрофон и соответствующие библиотеки. Инструкции по установке и запуску:

Python
bash
pip install pyaudio numpy scipy
python tuner_python.py
C++
bash
# Требуется PortAudio и FFTW (или Qt для GUI)
g++ -std=c++17 tuner_cpp.cpp -o tuner -lportaudio -lfftw3
./tuner
Java
bash
# Требуется JavaSound (встроена)
javac tuner_java.java && java tuner_java
C# (.NET Core)
bash
dotnet add package NAudio
dotnet run
Go
bash
# Требуется PortAudio (установите порт)
go get github.com/gordonklaus/portaudio
go run tuner_go.go
Rust
bash
cargo add cpal
cargo run
JavaScript (Node.js)
bash
# Требуется Web Audio API (работает в браузере)
# Откройте index.html в браузере

📦 Содержимое репозитория

Поддержка альтернативных строев (Drop D, Open G, DADGAD и др.).

Запись и воспроизведение для анализа.

Темперамент (равномерно-темперированный строй).
